# 语义映射参考

## 目录映射表

| 语义类型 | 目录路径 | 关键词 |
|----------|----------|--------|
| character | 设定/人物档案/ | 角色、人物、主角、配角、char |
| worldview | 设定/世界观/ | 世界观、世界、设定、规则、体系 |
| organization | 设定/组织势力/ | 组织、势力、门派、宗门、家族、org |
| relationship | 设定/关系网络.md | 关系、人物关系、关系网 |
| foreshadow | 状态追踪/伏笔管理/ | 伏笔、铺垫、fs、悬念 |
| timeline | 状态追踪/时间线.md | 时间线、时间、事件顺序 |
| state | 状态追踪/当前状态.md | 当前状态、角色状态、状态快照 |
| outline | 大纲/ | 大纲、章节大纲、故事线、ch |
| scene | 场景设计/ | 场景、分场、sc |
| content | 正文/ | 正文、章节内容、第N章 |
| project | 项目信息/ | 项目、配置、宪法、创作 |

## 查询示例

```
--query "林叶的角色档案"
--query "配角有哪些"
```

```

```

```
--query "天澜宗的信息"
```


## ID前缀规范

| 类型 | ID格式 | 示例 |
|------|--------|------|
| 角色 | char_XXX | char_001 |
| 组织 | org_XXX | org_001 |
| 章节 | ch_XXX | ch_010 |
| 场景 | sc_XXX_YY | sc_010_01 |
| 伏笔 | fs_XXX | fs_001 |

## 匹配优先级

1. **ID精确匹配** (权重: 10)
2. **名称匹配** (权重: 8)
3. **标签匹配** (权重: 5)
4. **摘要匹配** (权重: 2)

匹配结果按权重排序，返回最相关的结果。
