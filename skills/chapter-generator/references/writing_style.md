# 写作风格配方

## 默认写作风格配方

```
---
【✍️ 写作风格配方 — 像人类作家一样写】
---

🎯 句子节奏：动作戏短句连击(5-15字) → 心理戏长句一拉(带逗号断点) → 偶尔单句成段制造停顿

🎯 感官即动作：感官细节必须从角色动作或情节事件中自然产生——角色碰到什么才写触觉，闻到什么才写嗅觉。禁止为了"丰富画面"凭空插入感官描写段。
   ✓ "他扶住墙，指腹蹭到粗糙的灰浆" — 触觉来自动作
   ✓ "锅底烧焦了，满屋糊味，她咳了一声没理" — 嗅觉来自事件
   ✓ "筷子夹起那块肉时手抖了一下——凉的" — 触觉服务于心理
   ✗ "空气中弥漫着淡淡的血腥味" — 无来由的嗅觉装饰
   ✗ "一阵刺骨的寒风吹过他的脸颊" — 模板化触觉开场
   ✗ "远处传来隐隐的钟声/鸟鸣" — 万能听觉背景板

🎯 情绪表达：角色情绪只通过动作和生理反应透露(攥拳、咬牙、喉结滚动)，不写"他感到/意识到"

🎯 对话动作化：对话前后嵌入小动作("她别过脸，'走吧'")，不用"说道/开口道"等标签

🎯 开头多样化：连续段落不以相同词开头，用环境/动作/对话/声音交替起句

🎯 收尾留白：章节结尾用动作定格或悬念引钩，不做总结性抒情

🎯 比喻克制：全章比喻≤3处，每个比喻必须服务于情绪或剧情，拒绝装饰性比喻
```

## 写作红线

### 零容忍词汇（禁止使用）

| 词汇 | 原因 |
|------|------|
| 不禁 | 滥用程度极高，AI痕迹明显 |
| 此刻 | 模板化时间标记 |
| 眼眸 | 文艺腔过重 |
| 低眸 | 同上 |
| 勾唇 | 滥用表情描写 |
| 薄唇轻启 | 滥用表情描写 |
| 宛如 | 滥用比喻词 |
| 仿若 | 同上 |

### 限频词汇（每百字最多1次）

| 词汇 | 替代方案 |
|------|----------|
| 然而 | 但、可、却 |
| 突然 | 直接描写动作 |
| 缓缓 | 删掉或用具体动作 |
| 微微 | 删掉或用具体描写 |
| 轻轻 | 删掉或用具体描写 |
| 淡淡 | 删掉或用具体描写 |

## 节奏等级指导

### 等级1：极慢
- 更多环境/心理描写
- 减少事件密度
- 重氛围铺陈
- 适用于：铺垫章节、情感章节

### 等级2：较慢
- 以描写与情绪推进为主
- 事件推进适中
- 适用于：过渡章节、伏笔埋设

### 等级3：适中
- 描写与事件推进均衡
- 保持清晰转折
- 适用于：常规章节

### 等级4：较快
- 以动作与冲突推进为主
- 减少冗长铺垫
- 适用于：冲突升级章节

### 等级5：极快
- 高密度事件与转折
- 紧凑推进
- 避免长段描写
- 适用于：高潮章节、爽点爆发

## 场景设计原则

1. **单个场景内不要把冲突完全解决**
   - 留悬念给下一场景
   - 保持读者期待

2. **角色态度转变需要渐进**
   - 不能一个场景内从敌对变亲密
   - 每个场景只推进一小步

3. **action/conflict/outcome 用具体动作描述**
   - 不要用抽象形容（如"领悟了道理"）
   - 用可观察的行为表现

## 对话写作规范

### 禁止的对话标签
- 说道
- 问道
- 开口道
- 回答道

### 推荐的对话写法

```markdown
# 错误示范
"你来了。"他说道。
"是的。"她回答道。

# 正确示范
"你来了。"他头也不抬，手里的茶杯轻轻磕在桌上。
"是的。"她站在门口，没往里走。
```

## 心理描写规范

### 禁止的写法
- "他感到一阵悲伤"
- "她意识到自己错了"
- "心中暗道不好"

### 推荐的写法
- 用动作表现：攥紧拳头、咬紧牙关
- 用生理反应：喉结滚动、手心出汗
- 用行为选择：转身离开、沉默不语

## 感官描写规范

### 必须依附于动作或事件

```markdown
# 错误示范
空气中弥漫着淡淡的血腥味，远处传来隐隐的钟声。

# 正确示范
他蹲下身，手指蹭过地上的暗红——铁锈味。远处钟楼响了三下。
```
